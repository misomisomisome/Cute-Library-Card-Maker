What's actually automated

Everything except the literal login screen on each platform. That gate is each platform's anti-bot mechanism — not an oversight — so freedeploy never scripts past it; it gets you to one command or one click, then the deploy itself is one line forever after.

If "no signup at all" matters most: GitHub Pages reuses the GitHub login you already have. Every push to main deploys automatically, with no extra account ever created.
